## 📁 File Structure

```
/
└── index.html        ← The entire website (all pages, images, styles & scripts)
└── README.md         ← This file
```

This is a **single-file website**. All three pages (Home, About, Store) are contained within `index.html` and rendered via JavaScript — no separate HTML files are needed. All images are embedded as base64, so the site works offline and requires no external image hosting.

## ✏️ How to Update

### Update a Selar product link
Search for the placeholder link (e.g. `https://selar.co/visual-story-core-edition`) in `index.html` and replace with your real Selar URL.

### Update Microsoft Forms survey links
Search for `https://forms.microsoft.com` in `index.html` — there are 2 instances. Replace each with your real survey URLs.

### Update your photo
The photos are embedded as base64 in the HTML. To swap a photo, re-generate the file with the new image using the same build process.

### Add a testimonial
Find the `testi-grid` section in `index.html` and copy one of the existing `testi-card` blocks, then edit the text, author name and product name.

---

## 🚀 Deployment (GitHub Pages)

1. Ensure the main file is named **`index.html`**
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch → main → / (root)**
4. Save — your site will be live in ~1 minute
